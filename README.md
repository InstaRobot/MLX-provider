# MLX-provider

Provider for MLX LLM — Apple Silicon GPU-accelerated machine learning inference.

## Overview

This repository provides a provider integration for running LLM workloads on Apple Silicon using the MLX framework.

## Requirements

- Apple Silicon Mac (M1/M2/M3/M4)
- macOS 14+
- MLX framework installed

## Usage

```python
import mlx.core as mx
# Your MLX inference code here
```
